# SteamOS-Scripts

Collection of scripts to automate various things in SteamOS where gamescope "Game Mode" or the immutable nature of the OS cause better-known options to fail. 

To use any of these files, download them to your /home/deck folder on your steam deck, steam machine, or other SteamOS device. Double-click them or run them in a Konsole window. If running in Konsole, you may have to make them executable by doing chmod +x script_name.sh
